First open ~/.bashrc and add your aliases

My aliases:

alias sb='sublime-text.subl'
alias ccc='cd ..'
alias gpom='git push origin master'
alias gpul='git pull'
alias gpus='git push'
alias ga='git add -A'
alias gc='git commit -am'
alias sl='ls'
ls='ls -ac --color'

Finally use command source ~/.bashrc to see changes
